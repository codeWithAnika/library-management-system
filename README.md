# 📚 Library Management System

A Flask-based Library Management System developed to efficiently manage library operations such as book records, user management, searching, and reporting. This project demonstrates the practical application of Database Management System (DBMS) concepts and web development using Python.

---

## 🚀 Features

* User Registration and Login
* Dashboard for Library Management
* Add New Books
* View Available Books
* Search Books
* Generate Reports
* Responsive User Interface
* Custom Error Pages (404 & 500)

---

## 🛠️ Technologies Used

* Python
* Flask
* SQL Database
* HTML5
* CSS3
* Jinja2 Templates

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/codeWithAnika/library-management-system.git
cd library-management-system
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Configure the Database

Import the provided SQL file:

```text
LIBDB.sql
```

into your database management system.

### 6. Run the Application

```bash
python app.py
```
